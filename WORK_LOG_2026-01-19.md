## 5. Best Practices Learned

### Logo Files Must Be Committed
When adding new tools with logos:
1. Add logo files explicitly: `git add site/assets/logos/new_logo.png`
2. Run `git status` to verify logos are staged before committing
3. Any logo referenced in `tools.json` must exist in the commit
